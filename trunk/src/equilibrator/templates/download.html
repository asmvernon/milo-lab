Formats

eQuilibrator data is available for download in several formats.

  • JSON: lie JavaScript Object Notation format is a lightly structured hierarchical format that is easily parsed in most programming languages.
  • TSV: Tab-Separated Value format is a flat format that is mostly unstructured, easy to parse and easy to view in Excel and other spreadsheet programs. We use the SBTab dialect of TSV which helps provide some structure.

Compound Formation Energies
